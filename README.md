# 🧪 GenAI-Detection-Rules

이 저장소는 **악성코드 패턴(YARA)**, **웹공격 기반 네트워크 탐지(Snort)**, **행위 기반 로그 분석(Behavior)** 을 단계적으로 실습하기 위한 자료를 포함합니다.

```
practice1_yara/       # 악성코드·로더 샘플 기반 YARA 탐지 실습
practice2_snort/      # CVE 기반 웹공격 PCAP으로 Snort 룰 작성 실습
practice3_behavior/   # 웹로그·시스템로그를 통한 행위 기반 탐지 실습
```

각 폴더에는 **실습용 샘플 + 내부정보 기반 공격 흐름 + 탐지포인트 설명**이 포함되어 있습니다.
